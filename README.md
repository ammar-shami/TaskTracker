# Task Tracker CLI

A simple command-line task tracker application made in C++.

## Features

- Add Tasks
- View Tasks
- Complete Tasks
- Delete Tasks
- Save Tasks in File

## Technologies Used

- C++
- File Handling
- Vectors

## How to Compile

g++ main.cpp -o tasktracker

## How to Run

Windows:
tasktracker.exe

Linux/Mac:
./tasktracker
